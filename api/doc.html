SSC API Documentation

This document provides an overview of the endpoints available in the SSC API for managing questions and categories, including request methods, content types, and example requests and responses.

Category Management

GET /api/category.php

Request

Method: GET

Content-Type: application/json

Example: GET /api/category.php?id=1
Example: GET /api/category.php?keyword=general

Response

Content-Type: application/json

          {
            "status": 200,
            "message": "Data fetched successfully",
            "data": [
                {
                    "id": "1",
                    "category_name": "General Knowledge",
                    "image": ""
                }
            ]
          }

POST /api/category.php

Request

Method: POST

Content-Type: application/json

          {
            "category_name": "General Knowledge", *
            "image": ""
          }

Response

Content-Type: application/json

          {
            "status": 200,
            "message": "Category created successfully",
            "data": []
          }

PUT /api/category.php?id=1

Request

Method: PUT

Content-Type: application/json

          {
            "category_name": "General Knowledge", *
            "image": ""
          }

Response

Content-Type: application/json

          {
            "status": 200,
            "message": "Category updated successfully",
            "data": []
          }

DELETE /api/category.php?id=1

Request

Method: DELETE

Content-Type: application/json

          {
            "id": 1 *
          }

Response

Content-Type: application/json

          {
            "status": 200,
            "message": "Category deleted successfully",
            "data": []
          }

Question Management

GET /api/question.php

Request

Method: GET

Content-Type: application/json

Example: GET /api/question.php?id=1

Response

Content-Type: application/json

          {
            "response": [
              {
                  "id": "1",
                  "ssc_category": "General Knowledge",
                  "question": "What is the capital of Germany?",
                  "optiona": "Berlin",
                  "optionb": "Madrid",
                  "optionc": "Paris",
                  "optiond": "Rome",
                  "answer": "a",
                  "note": "",
                  "image": ""
              }
            ],
            "status": 200,
          }

POST /api/question.php

Request

Method: POST

Content-Type: application/json

          {
            "ssc_category": "General Knowledge", *
            "question": "What is the capital of Germany?", *
            "optiona": "Berlin", *
            "optionb": "Madrid", *
            "optionc": "Paris", *
            "optiond": "Rome", *
            "answer": "a", *
            "note": "Berlin is the capital of Germany.",
            "image": ""
          }

Response

Content-Type: application/json

          {
            "response": {
                "status": 200,
                "message": "Question created successfully"
            },
            "status": 200
          }
