Category API

GET /category.php

Fetch categories based on query parameters.

Request Parameters:

  • id: (optional) Category ID.
  • type: (optional) Category type.
  • keyword: (optional) Search keyword for category name.
  • table: (optional) If set, fetch paginated results.
  • page: (optional) Page number for paginated results.
  • limit: (optional) Number of records per page.

Example Request:

GET /category.php?type=1&keyword=Science

Example Response:

{
  "status": 200,
  "message": "Data fetched successfully",
  "data": [
    {
      "id": 1,
      "category_name": "Science",
      "type": 1,
      "questions": 10,
      "total_duration": 600000
    }
  ]
}

POST /category.php

Create a new category.

Request Body:

{
  "category_name": "Mathematics",
  "type": 2,
  "image": "math.png",
  "instructions": "Solve the following problems|Answer correctly"
}

Example Response:

{
  "status": 200,
  "message": "Category created successfully",
  "data": []
}

PUT /category.php?id=1

Update an existing category by ID.

Request Body:

{
  "category_name": "Physics",
  "image": "physics.png",
  "instructions": "Follow instructions|Complete all tasks"
}

Example Response:

{
  "status": 200,
  "message": "Category updated successfully",
  "data": []
}

DELETE /category.php

Delete a category by ID.

Request Body:

{
  "id": 1
}

Example Response:

{
  "status": 200,
  "message": "Category deleted successfully",
  "data": []
}

Question API

GET /question.php

Fetch questions based on query parameters.

Request Parameters:

  • id: (optional) Question ID.
  • category: (optional) Category ID.
  • type: (optional) Category type.
  • table: (optional) If set, fetch paginated results.
  • page: (optional) Page number for paginated results.
  • limit: (optional) Number of records per page.

Example Request:

GET /question.php?category=1&type=1

Example Response:

{
  "status": 200,
  "response": [
    {
      "id": 1,
      "question": "What is the capital of France?",
      "optiona": "Paris",
      "optionb": "Berlin",
      "optionc": "Madrid",
      "optiond": "Rome",
      "answer": "Paris",
      "duration": 60000,
      "category_name": "Geography",
      "type": 1
    }
  ]
}

POST /question.php

Create a new question.

Request Body:

{
  "category_id": 1,
  "question": "What is 2 + 2?",
  "optiona": "3",
  "optionb": "4",
  "optionc": "5",
  "optiond": "6",
  "answer": "4",
  "duration": 1
}

Example Response:

{
  "status": 200,
  "message": "Question created successfully"
}

PUT /question.php?id=1

Update an existing question by ID.

Request Body:

{
  "question": "What is 3 + 3?",
  "answer": "6",
  "duration": 2
}

Example Response:

{
  "status": 200,
  "message": "Question updated successfully"
}

DELETE /question.php

Delete a question by ID.

Request Body:

{
  "id": 1
}

Example Response:

{
  "status": 200,
  "message": "Question deleted successfully"
}

PDF API

GET /pdf.php

Fetch PDFs based on query parameters.

Request Parameters:

  • id: (optional) Category ID.
  • type: (optional) Category type.
  • keyword: (optional) Search keyword for PDFs.
  • table: (optional) If set, fetch paginated results.
  • page: (optional) Page number for paginated results.
  • limit: (optional) Number of records per page.

Example Request:

GET /pdf.php?type=1&keyword=Science

Example Response:

{
  "status": 200,
  "data": [
    {
      "id": 1,
      "category_name": "Science",
      "pdf": "http://localhost/uploads/currentaffairs/science.pdf",
      "type": 1
    }
  ],
  "message": "PDF fetched successfully"
}

POST /pdf.php

Create a new PDF entry.

Request Body:

{
  "pdf": "http://localhost/uploads/currentaffairs/newfile.pdf",
  "type": 2
}

Example Response:

{
  "status": 200,
  "message": "PDF added successfully"
}

PUT /pdf.php?id=1

Update an existing PDF entry by ID.

Request Body:

{
  "pdf": "http://localhost/uploads/currentaffairs/updatedfile.pdf"
}

Example Response:

{
  "status": 200,
  "message": "PDF updated successfully"
}

DELETE /pdf.php

Delete a PDF entry by ID.

Request Body:

{
  "id": 1
}

Example Response:

{
  "status": 200,
  "message": "PDF deleted successfully"
}
